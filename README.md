# Ambifo

A modern and responsive corporate website built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**.

## 🚀 Features

* Responsive design
* Modern UI
* Multiple service and solution pages
* Fast performance with Vite

## 🛠️ Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS

## 📦 Getting Started

```bash
git clone https://github.com/Vinit010/Ambifo.git
cd Ambifo
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
```
